## 2. 建议位置与核心技能树
基于您的身体条件和现有风格，建议**强化3D锋线**定位：

1. **接球投篮稳定性** - 三分线外接球即投命中率

## 3. 一周训练计划（每周4小时，分3次）

### **周一：投篮与无球训练（75分钟）**
- **热身**：动态拉伸 + 运球绕场（10分钟）
- **接球投篮**：三分线5个点，每点10次接球即投（25分钟）
- **移动投篮**：底线-肘区-底线跑动接球投篮（15分钟）
- **空切练习**：模拟V-cut、L-cut接球上篮（15分钟）
- **冷却**：静态拉伸 + 罚球练习（10分钟）

### **周三：持球与终结训练（75分钟）**
- **热身**：敏捷梯 + 运球组合（10分钟）
- **一运终结**：接球后一次运球完成上篮/跳投（20分钟）
- **三威胁**：假投真突、假突真投组合（15分钟）
- **对抗上篮**：用护具模拟对抗完成终结（15分钟）
- **冷却**：核心力量训练（平板支撑等）（15分钟）

### **周五：防守与实战模拟（90分钟）**
- **热身**：防守滑步 + 反应训练（15分钟）
- **closeout防守**：快速接近并封堵投篮（20分钟）
- **协防轮转**：2v3防守轮转练习（20分钟）
- **半场3v3**：应用所学技能（25分钟）
- **冷却**：全身拉伸（10分钟）

## 4. 战术建议与实战注意事项

### **战术建议：5-out（五外站位）**
最适合您当前风格的战术：
- **站位**：您站在弱侧45度三分线
- **执行**：持球人突破时，您根据防守选择：
  1. 防守收缩 → 向外一步接球投篮
  2. 防守跟防 → 向篮下空切
  3. 防守失位 → 接球后一运中距离
- **优势**：最大化您的投射威胁，简化决策

### **业余局实战注意事项（3条）：**

1. **先做好防守再想进攻**
   - 业余比赛防守赢得信任
   - 专注盯防对方得分手，积极协防
   - 每次防守回合都大声沟通

2. **简化进攻选择**
   - 接球后决策不超过2秒
   - 有空位果断投篮，没机会立即传球
   - 减少不必要的运球，多一次传球

3. **保持体力分配**
   - 前两节：70%防守 + 30%进攻（建立防守存在感）
   - 第三节：50%防守 + 50%进攻（寻找得分机会）
   - 第四节：根据比分调整，关键时刻敢于出手

### **额外建议：**
- **每周加练**：额外30分钟纯罚球训练（提升罚球命中率）
- **录像分析**：每月录一次自己比赛，重点看无球跑动和防守选择
- **力量训练**：每周1-2次下肢力量训练（深蹲、弓步蹲），提升对抗稳定性
